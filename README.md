# How to build the project on Mac
## Env setup
install ruby@2.7
```zsh
brew install ruby@2.7
chruby_use /usr/local/opt/ruby@2.7/
```
or
```zsh
rbenv install 2.7.3
rbenv local 2.7.3
```

## Run
```zsh
bundle install && bundle exec jekyll serve --incremental
```
